# Changelog

All notable changes to TimeGlance will be documented in this file.

## 0.1.0 - 2026-06-21

- Added runtime and memory forecasting for synchronous iterables.
- Added async iterable support with `aforecast`.
- Added `ForecastSession` and `forecast_function` helpers.
- Added JSON, CSV, and HTML report exporters.
- Added modern PyPI packaging metadata and tests.
